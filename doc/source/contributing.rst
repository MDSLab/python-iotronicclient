===================================
Contributing to python-iotronicclient
===================================

If you're interested in contributing to the python-iotronicclient project,
the following will help get you started.

#openstack-iotronic on Freenode IRC Network
-----------------------------------------
There is a very active chat channel at irc://freenode.net/#openstack-iotronic.
This is usually the best place to ask questions and find your way around.
IRC stands for Internet Relay Chat and it is a way to chat online in real
time. You can ask a question and come back later to read the answer in the
log files. Logs for the #openstack-iotronic IRC channel are stored at
http://eavesdrop.openstack.org/irclogs/%23openstack-iotronic/.

Contributor License Agreement
-----------------------------

.. index::
   single: license; agreement

In order to contribute to the python-iotronicclient project, you need to have
signed OpenStack's contributor's agreement.

.. seealso::

   * http://docs.openstack.org/infra/manual/developers.html
   * http://wiki.openstack.org/CLA

LaunchPad Project
-----------------

Most of the tools used for OpenStack depend on a launchpad.net ID for
authentication. After signing up for a launchpad account, join the
"openstack" team to have access to the mailing list and receive
notifications of important events.

.. seealso::

   * http://launchpad.net
   * http://launchpad.net/python-iotronicclient
   * http://launchpad.net/~openstack
